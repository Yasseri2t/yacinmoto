# ScootZone

A Laravel e-commerce app for scooter parts and accessories.

## Tech Stack

- **Laravel 12** (PHP 8.2+)
- **SQLite** database (zero-config, perfect for single-server deploy)
- **Tailwind CSS v4** + Vite
- **Render.com** for hosting

## Local Setup

```bash
git clone https://github.com/YOUR_USERNAME/scootzone-app.git
cd scootzone-app

composer install
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan storage:link
npm install && npm run build

php artisan serve
```

Visit `http://localhost:8000`

## Admin Panel

Go to `/admin/login` and enter the password set in your `.env` as `ADMIN_PASSWORD`.

## Deploy to Render.com

1. Push this repo to GitHub
2. Go to [render.com](https://render.com) → New → Web Service
3. Connect your GitHub repo
4. Render auto-detects `render.yaml` — just click **Deploy**
5. Set `ADMIN_PASSWORD` in Render's Environment Variables dashboard

## Environment Variables (required on Render)

| Variable | Description |
|---|---|
| `APP_KEY` | Auto-generated by Render |
| `ADMIN_PASSWORD` | Your admin panel password |
| `APP_URL` | Your Render app URL |
